📖 知识库 · CPA 六科

参照 CPA 专业阶段六门科目整理面试常考知识点。当前为初步框架，后续逐步完善。

三张财务报表的勾稽关系 ▶
核心逻辑：利润表的净利润 → 进入资产负债表的留存收益；同时影响现金流量表的经营活动现金流。

面试高频问题：
① 如果一家公司收入增加但现金流减少，可能是什么原因？
② 如何从三张表中判断一家公司是否存在盈余管理嫌疑？
③ 资产负债表不平，如何排查？
收入确认五步法（新收入准则） ▶
识别合同 → 识别履约义务 → 确定交易价格 → 分摊价格 → 在满足履约义务时确认收入。

重点：时段 vs 时点确认、可变对价估计、重大融资成分的处理。
金融资产分类与计量 ▶
三类：① 摊余成本计量（AC）② 公允价值计量且变动计入其他综合收益（FVOCI）③ 公允价值计量且变动计入当期损益（FVTPL）

分类依据：业务模式 + 合同现金流量特征。
长期股权投资与合并报表 ▶
控制（→ 成本法）vs 重大影响（→ 权益法）vs 无控制无重大影响 → 金融资产
商誉 = 合并成本 - 可辨认净资产公允价值份额
审计风险模型 ▶
审计风险 = 重大错报风险 × 检查风险

重大错报风险 = 固有风险 × 控制风险（与被审计单位相关）
检查风险 = 与审计程序设计和执行相关（审计师可以控制）
审计程序类型 ▶
七大类：检查、观察、询问、函证、重新计算、重新执行、分析程序

函证重点：银行函证（100%发函）、应收账款函证，控制函证全过程。
三大审计报告意见类型 ▶
① 无保留意见 ② 保留意见 ③ 否定意见 ④ 无法表示意见
杜邦分析体系 ▶
ROE = 净利率 × 资产周转率 × 权益乘数
三层拆解：经营 profitability、资产效率、财务杠杆。
DCF 估值模型 ▶
企业价值 = Σ FCFFₜ / (1+WACC)ᵗ + 终值
FCFF = 息税前利润×(1-税率) + 折旧摊销 - 资本性支出 - 营运资本增加
本量利分析 ▶
盈亏平衡点销量 = 固定成本 / (单价 - 单位变动成本)
经营杠杆系数 DOL = 边际贡献 / 息税前利润
增值税核心要点 ▶
税率：13%（基本）、9%、6%、0%
计税方法：一般计税（销项-进项）、简易计税
企业所得税 ▶
基本税率 25%，高新企业 15%
广宣费 15%、业务招待费 60%/5‰、公益性捐赠 12% 等扣除限额
公司法·公司治理结构 ▶
三会一层：股东会、董事会、监事会、经理层
上市公司独立董事制度：至少 1/3 为独立董事
证券法·信息披露 ▶
信息披露三原则：真实、准确、完整
内幕交易、操纵市场、虚假陈述的法律责任
波特五力模型 ▶
五力：供应商议价能力、购买者议价能力、新进入者威胁、替代品威胁、现有竞争者强度
SWOT 与 BCG 矩阵 ▶
SWOT：内部优势/劣势 + 外部机会/威胁
BCG：明星、金牛、问题、瘦狗
风险管理 ▶
COSO 框架：内控五要素
风险管理策略：承担、规避、转移、转换、对冲、控制
