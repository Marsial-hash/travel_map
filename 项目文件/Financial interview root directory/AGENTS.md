# AGENTS.md — AI 助手说明书

## 项目概述

这是一个**金融面试全流程备考平台**（Financial Interview），由一名财务管理专业学生个人开发。项目目标是帮助财务/审计方向的应届生完成从简历准备到面试实战的全流程。

## 项目负责人

- **背景**：财务管理专业在读学生
- **技术能力**：学生水平，建议使用轻量级技术方案
- **开发角色**：独立开发者，一个人完成产品设计、开发、部署全流程

## 核心功能（PRD 已确认）

1. **简历押题（简历 + 模拟面试合并）**
   - 上传简历（PDF / Word / 图片 → OCR 自动解析填入各字段）
   - 字段：姓名、联系方式、教育经历、实习经历、项目经历、技能证书、自我评价
   - AI 自动生成预测面试题，按行为面 / 简历面 / 专业面分类展示
   - 点击任一题目 → 进入对话式练习（面试官 ↔ 候选人，Enter 发送）
   - 支持编辑简历后重新生成题目
3. **面经库**
   - 支持 Word (.docx) / PDF / 图片 (PNG/JPG) 导入
   - 自动提取关键词：公司名、岗位、面试轮次（群面/经理面/par面/return面等）
   - 多维度筛选：按公司 / 轮次 / 岗位 筛选
   - 搜索策略：优先精确匹配（标题/公司/岗位/轮次）→ 无结果则模糊匹配正文内容
   - 搜索结果高亮显示关键词
4. **面试日历 & 进度管理**
   - 四栏看板：已投递 → 笔试 → 面试 → 已结束
   - 可添加/编辑/删除投递记录
   - 一键切换阶段
   - 支持标注日期、备注
5. **知识库**（参照 CPA 6 科：会计、审计、财管、税法、经济法、战略 — 后续完善）

## 不做

- 社区/社交功能（面经共享、问答、伙伴匹配）

## 与 AI 协作约定

1. **同步更新**：用户每次提出新要求或做出决定，主动同步到 AGENTS.md 和 CLAUDE.md，以及对应的 docs/notes/ 文档
2. **保持简洁**：文档不写废话，只记录有用信息
3. **技术选型**：倾向轻量、免费、低维护成本方案
4. **个人项目**：不涉及团队协作、CI/CD
