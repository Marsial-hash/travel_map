# CLAUDE.md — 项目开发规则

## 技术栈

- **前端**：HTML / CSS / JavaScript（原生为主，初期避免构建工具）
- **后端**：localStorage 本地存储优先；如需云服务考虑 Supabase
- **部署**：GitHub Pages 或 Vercel
- **版本管理**：Git

## 开发约定

1. **中文优先**：UI 界面、注释、文档均使用中文
2. **轻量优先**：不引入不必要的框架、库或构建工具
3. **渐进增强**：先做静态可用的版本，再逐步增加功能
4. **移动端适配**：优先考虑移动端浏览体验（学生多在手机上操作）

## 代码风格

- HTML：语义化标签，避免过度嵌套
- CSS：使用 CSS 变量管理主题色，命名采用 BEM 或语义化 class
- JavaScript：ES6+ 语法，不依赖 jQuery
- 文件名：英文小写 + 连字符

## 项目阶段

1. **Phase 1 — PRD & 设计**：确定功能范围，出站点地图和原型（当前阶段 ✓ Demo v0.3 已完成）
2. **Phase 2 — MVP 开发**：面经库 + 简历模板 + 面试题库 + 面试日历
3. **Phase 3 — 迭代优化**：根据使用反馈补充功能
